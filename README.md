1. Addition
2. Subtraction
3. Multiplication
4. Division

## Requirements:
Python 3.x

## Installation: 
Download python for your system https://www.python.org/

## Run Project:
1. Clone the repository
2. Open the terminal
3. Navigate to the project
4. Run the following command to start the project

````````````````````````````````````````````````````````````
python3 code/main.py
````````````````````````````````````````````````````````````
